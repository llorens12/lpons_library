static

Arguments

$data

\mysqli

Description: contains all data to show in the table.

$edit

boolean|false

Description: if is true, add column edit, else not.

$drop

boolean|false

Description: if is true, add column drop, else not.

$add

boolean|false

Description: if is true, add column add, else not.

$objectAdd

string

Description: contains the url to redirect in column add. (He include controller.php?method=showAdd{this include your url})

$typeObject

string

Description: contains the url to redirect in the columns edit and drop. (In edit he include: controller.php?method=showEdit{this include your url}) and (In drop he include: controller.php?delete=setDelete{this include your url}).

$primaryKeys

array|string

Description: this array contains the names of the columns that are primary keys (Example: ["user","Books","ID Copy",...])

$reserveDelimiter

boolean|false

Description: in the case of reserves, if the reserve is sent you can't delete and is received nor can edit reserve. Select true if you want this option. (NOTE: this option not delete the columns sent and received, this is the next option)

$showColumnSentReceived

boolean|false

Description: select true if you don't want show columns sent and true.

$MAX_DAYS_RESERVE

integer

Description: contains the max days of reserve.

$sid

Description: this contains the session id.

Response

string

Description: return the table.
