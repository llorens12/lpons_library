index:
	-inicia session como anonimous (los demas objetos si la conexion recibida es anonimous redirecciona a Anonimous)
	-redirige a Anonimous

Anonimous:
	-permite iniciar session
	-muestra el login
	-muestra register
	-permite registrar un nuevo usuario

User:
	-realiza log out
	-redirecciona a la home de cada usuario (añadir campo home a la tabla usuarios)
	-permite configurar la cuenta
	-Es donde se realizan todas las acciones que no estan relacionadas con el content

Controller:
	-se encarga de todo lo que este relacionado con la base de datos
	-le pasamos un churro de datos y el desfragmenta el churro y lo inserta en la base de datos
	-es una classe abstracta

Theme:
	-es la base de la apariencia, es la que se encarga de mostrar el contenido.

CommonStyles:
	-contiene todo lo que sea graficamente comun entre los objetos.
